Introduction

Everyone saw different interesting robots which can perform really interesting things. However does anyone really asked himself a question: "How?". ROS

What is ROS?

ROS stands for Robotic Operating System. It is designed to completely fullfil one's desire to control robots. It provides a set of package to control robot, so, we can just install it and run our robots as we want.

Why ROS?

ROS is higly developed robotic System, also because of its open-source philosophy it is easy to get into and a lot of projects are done not only by professional but by enthusiast too.
